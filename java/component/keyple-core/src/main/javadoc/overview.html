The role of the Keyple Core module is the management of communications with the secure elements (SE) using smart card readers.

It implements plugins in charge of abstracting readers with or without contact through a generic API (seproxy package).
Keyple Core also provides classes for selecting and transmitting commands to SEs in a generic way with optional built-in observation mechanisms (selection, command and event packages).
In addition Keyple Core also provides utility classes for data formatting or analysis (util package).
